## install

1. env.example ni .env qilinadi
2. DB ulanadi
3. Quidagi kammandalari beriladi:

```bash
$ composer install
```

```bash
$ php artisan key:generate
```

```bash
$ php artisan migrate
```

```bash
$ php artisan db:seed
```
